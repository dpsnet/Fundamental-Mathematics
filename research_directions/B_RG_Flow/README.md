## 2. 核心数学框架

### 2.1 有效维度作为尺度函数

**定义**: 对于度量空间 $(X, d)$，**有效维度**在尺度 $r$：
$$d_{\text{eff}}(r) = \frac{\log N(r)}{\log(1/r)}$$

其中 $N(r)$ 是覆盖数。

### 2.2 函数方程理论

**核心问题**: $d_{\text{eff}}(r)$ 满足什么函数方程？

**自相似情况**:
对于严格自相似分形，有：
$$d_{\text{eff}}(\lambda r) = d_{\text{eff}}(r), \quad \lambda = \text{相似比}$$

**推广**: 尺度依赖的函数方程：
$$d_{\text{eff}}(r) = f(d_{\text{eff}}(\phi(r)), r)$$

## 4. 关键定理目标

### 定理 B.1: 自相似函数方程
对于相似比 $r_1, \ldots, r_N$ 的自相似分形：
$$d_{\text{eff}}(r) = \frac{\log(\sum_{i=1}^N d_{\text{eff}}(r/r_i)^{-1})}{\log(1/r)}$$

**证明思路**: 覆盖数的自相似性。

### 定理 B.2: 固定点存在性
在适当函数空间中，不动点方程有唯一解 $d^*$。

**证明思路**: Banach 不动点定理。

### 定理 B.3: M-0.7 "谱维流动" 的严格解释
M-0.7 的流动方程可以严格化为 **函数方程的迭代**。
