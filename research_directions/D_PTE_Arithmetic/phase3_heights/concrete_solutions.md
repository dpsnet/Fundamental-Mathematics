# PTE 理想解的具体实例分析

## 已知解的高度统计

---

## 1. 理想解的定义回顾

$P = (a_1,\ldots,a_n; b_1,\ldots,b_n)$ 是理想解，如果：
$$\sum_{i=1}^n a_i^k = \sum_{i=1}^n b_i^k, \quad k = 1, 2, \ldots, n$$

等价于：多重集 $\{a_i\}$ 和 $\{b_i\}$ 的前 $n$ 个幂和相等。

---

## 2. 经典解的高度计算

### 2.1 Prouhet 解 (n = $2^m$)
**构造**: 使用 Prouhet-Thue-Morse 序列。

对于 $m=1$ ($n=2$)：
- 解：$(0, 3; 1, 2)$
- 高度：$H = 3$

对于 $m=2$ ($n=4$)：
- 解：$(0, 3, 5, 6; 1, 2, 4, 7)$
- 验证：
  - $k=1$: $0+3+5+6 = 14 = 1+2+4+7$ ✓
  - $k=2$: $0+9+25+36 = 70 = 1+4+16+49$ ✓
  - $k=3$: $0+27+125+216 = 368 = 1+8+64+343$ ✓
  - $k=4$: $0+81+625+1296 = 2002 = 1+16+256+2401$ ✓
- 高度：$H = 7$

对于 $m=3$ ($n=8$)：
- 解：$(0, 3, 5, 6, 9, 10, 12, 15; 1, 2, 4, 7, 8, 11, 13, 14)$
- 高度：$H = 15$

**规律**: Prouhet 解的高度 $H = 2^{m+1} - 1 = 2n - 1$

### 2.2 非 Prouhet 解 (n = 4)
Letac (1942) 发现：
- $(0, 11, 13, 24; 1, 6, 18, 23)$
- 高度：$H = 24$

对比：Prouhet $n=4$ 解高度为 7，这是次优的。

### 2.3 n = 5 (无解)
**定理**: 不存在 $n=5$ 的理想解。

这是关于理想解存在性的深刻结果。

### 2.4 n = 6 (第一个非平凡理想解)
Borwein 等发现 (1990s)：
- $(0, 19, 25, 57, 62, 86; 2, 11, 40, 42, 69, 85)$
- 高度：$H = 86$

这是一个重要发现，因为 $n=6$ 的解非常稀少。

Prouhet 解：$H_{Prouhet}(n) = 2n - 1$（线性增长）

**猜想**: 对于非 2 的幂次 $n$，最小高度随 $n$ 指数增长。

## 4. 高度下界理论

### 4.1 代数约束
设 $P = (a_1,\ldots,a_n; b_1,\ldots,b_n)$ 是理想解。

**Newton 恒等式**: 幂和 $S_k$ 与初等对称多项式 $e_k$ 的关系。

对于理想解，$S_k(a) = S_k(b)$ ($k=1,\ldots,n$)。

### 4.2 高度下界
**猜想 (高度下界)**: 对于 $n$ 不是 2 的幂次：
$$H_{min}(n) \geq c \cdot e^{\alpha n}$$

对于 $n = 2^m$：
$$H_{min}(n) \leq 2n - 1$$

### 4.3 与 BSD 猜想的联系
$X_{n,n}$ 作为代数簇，其有理点与 L-函数的关系：
- 秩与解的存在性
- Sha 群与高度的精细结构

**注**: 这是高度推测性的联系，需要进一步研究。

## 5. 计数函数估计

### 5.1 定义
$$N_n(B) = \#\{n\text{-理想解} : H(P) \leq B\}$$

### 5.2 渐近猜想
**猜想 A**: 对于 $n = 2^m$：
$$N_n(B) \sim C_n \cdot B^{\alpha_n}, \quad \alpha_n > 0$$

**猜想 B**: 对于 $n$ 不是 2 的幂次且存在解：
$$N_n(B) \sim C_n \cdot (\log B)^{\beta_n}$$

（解非常稀少，多项式增长太慢）

### 5.3 与 Manin 猜想的联系
对于 Fano 簇，有理点计数：
$$N(B) \sim C \cdot B^a (\log B)^b$$

PTE 簇不是 Fano，但类似的计数原则可能适用。

## 6. 计算搜索策略

### 6.1 搜索空间剪枝
利用约束：
- $\sum a_i = \sum b_i = 0$（平移后）
- $\sum a_i^2 = \sum b_i^2$
- 对称性约化

### 6.2 并行搜索算法
对于给定 $n$ 和高度上界 $H_{max}$：
1. 枚举 $[-H_{max}, H_{max}]$ 中的 $n$ 元组
2. 快速筛选前几个幂和
3. 完全验证候选解

## 7. 与 M-0.9 的关系

### M-0.9 的错误声称
> "PTE 问题可以用分形方法解决，解有某种分布规律"

### 严格结果
- 解的"分布" = 代数簇 $X_{n,n}$ 上的有理点分布
- 高度函数 $H$ 提供复杂度度量
- 计数函数 $N_n(B)$ 描述密度

**结论**: 不需要分形，算术几何已提供充分框架。
