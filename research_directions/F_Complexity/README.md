## 2. 核心数学框架

### 2.1 BSS 模型

**定义**: 在实数域 $\mathbb{R}$ 上的计算：
- 输入: $(x_1, \ldots, x_n) \in \mathbb{R}^n$
- 操作: $+, -, \times, /, \leq$ 比较
- 复杂度: 操作次数

**复杂性类**:
- $P_\mathbb{R}$: 多项式时间可解
- $NP_\mathbb{R}$: 非确定性多项式时间
- $NP_\mathbb{R}$-完全问题存在 (如 4-FEAS)

### 2.2 几何问题的复杂性

**问题分类**:

1. **分形维数计算**:
   - 输入: 度量空间的描述
   - 输出: Hausdorff 维数
   - 复杂性: ???

2. **覆盖数计算**:
   - 输入: 集合 $S$，半径 $r$
   - 输出: 最小覆盖数 $N(r)$
   - 复杂性: 通常是 NP-难

3. **自相似分形识别**:
   - 输入: 点集数据
   - 输出: IFS 参数
   - 复杂性: ???

## 4. 关键定理目标

### 定理 F.1: 覆盖数计算的复杂性
给定紧集 $K \subset \mathbb{R}^n$ 和 $\epsilon > 0$，计算最小覆盖数是 **NP-难** 的。

**证明思路**: 从集合覆盖问题归约。

### 定理 F.2: 自相似维数计算的复杂性
对于 IFS 给出的自相似分形，计算 Hausdorff 维数在 **BSS 模型中是多项式时间** 的。

**证明思路**: 维数公式 $d$ 满足 $\sum r_i^d = 1$，可用牛顿法求解。

### 定理 F.3: M-0.20 "复杂度" 的严格解释
M-0.20 的"复杂度"可以严格化为 **计算复杂性类** 或 **Kolmogorov 复杂度**。
