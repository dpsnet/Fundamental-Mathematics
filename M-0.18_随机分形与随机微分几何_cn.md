## 1. 引言

### 1.1 随机分形的历史背景

随机分形的概念由Mandelbrot在1970年代提出，将分形几何与概率论相结合。与确定性分形不同，随机分形通过随机过程生成，具有更强的灵活性和对自然现象的描述能力。

经典的随机分形包括：
- **随机Cantor集**：通过随机删除区间构造
- **布朗运动轨迹**：具有Hausdorff维数2的随机分形曲线
- **随机Sierpinski垫片**：通过随机迭代函数系统生成

### 1.2 分形与随机过程的结合

分形结构与随机过程的结合产生了丰富的数学理论：

1. **分形布朗运动**：具有长程相关性的随机过程
2. **分形扩散**：在分形介质上的反常扩散
3. **随机谱维**：谱维的随机涨落模型
4. **分形随机微分方程**：描述分形结构上的随机动力学

这些理论为理解复杂系统的统计性质提供了数学工具。

## 2. 随机分形测度理论

### 2.1 随机迭代函数系统

**定义 2.1.1（随机迭代函数系统，RIFS）** 随机迭代函数系统是四元组$(X, \{f_i\}_{i=1}^N, \{p_i\}_{i=1}^N, (\Omega, \mathcal{F}, \mathbb{P}))$，其中：
- $X$是完备度量空间
- $\{f_i: X \to X\}$是压缩映射族，压缩比$L_i < 1$
- $\{p_i\}$是概率分布，$\sum_{i=1}^N p_i = 1$
- $(\Omega, \mathcal{F}, \mathbb{P})$是概率空间

**定义 2.1.2（随机压缩映射）** 随机压缩映射$f_{\omega}: X \to X$是满足以下条件的随机变量：

1. **压缩性**：$\mathbb{E}[d(f_{\omega}(x), f_{\omega}(y))] \leq L \cdot d(x, y)$，$L < 1$
2. **可测性**：$(\omega, x) \mapsto f_{\omega}(x)$是联合可测的

**定理 2.1.3（RIFS吸引子的存在性）** 设$(X, d)$是完备度量空间，$\{f_{\omega}\}$是随机压缩映射族，则存在唯一的紧随机集$K(\omega)$，使得：

$$K(\omega) = \bigcup_{i=1}^N f_{i,\omega}(K(T_i\omega))$$

其中$T_i: \Omega \to \Omega$是保测变换。

**证明**：定义算子$\mathcal{R}$在随机紧集空间上：
$$\mathcal{R}(A)(\omega) = \bigcup_{i=1}^N f_{i,\omega}(A(T_i\omega))$$

利用Hausdorff度量$h_d$和随机压缩条件，证明$\mathcal{R}$是压缩算子，应用Banach不动点定理。$\square$

### 2.2 随机自相似测度

**定义 2.2.1（随机自相似测度）** 测度族$\{\mu_{\omega}\}_{\omega \in \Omega}$称为随机自相似的，如果满足：

$$\boxed{\mu_{\omega} = \sum_{i=1}^N p_{i,\omega} \cdot (f_{i,\omega})_* \mu_{T_i\omega}}$$

其中$(f_{i,\omega})_*$表示推前测度。

**定理 2.2.2（随机自相似测度的存在唯一性）** 在适当的条件下，存在唯一的随机自相似测度$\mu_{\omega}$满足上述方程。

**证明**：定义Monge-Kantorovich度量：
$$W_1(\mu, \nu) = \inf_{\gamma \in \Gamma(\mu, \nu)} \int d(x, y) d\gamma(x, y)$$

在随机测度空间上，定义算子：
$$\mathcal{M}(\mu)(\omega) = \sum_{i=1}^N p_{i,\omega} \cdot (f_{i,\omega})_* \mu_{T_i\omega}$$

证明$\mathcal{M}$是压缩映射，应用Banach不动点定理。$\square$

**定理 2.2.3（随机测度的矩估计）** 随机自相似测度的$q$阶矩满足：

$$\mathbb{E}\left[\int |x|^q d\mu_{\omega}(x)\right] < \infty$$

当$\mathbb{E}[L_{\omega}^q] < 1$时成立。

### 2.3 随机分形维数

**定义 2.3.1（随机Hausdorff维数）** 随机集$K(\omega)$的Hausdorff维数定义为：

$$\dim_H K(\omega) = \inf\{s: \mathcal{H}^s(K(\omega)) = 0\} = \sup\{s: \mathcal{H}^s(K(\omega)) = \infty\}$$

**定理 2.3.2（随机维数的期望公式）** 在独立同分布条件下，随机Hausdorff维数几乎必然等于：

$$\dim_H K(\omega) = \frac{\mathbb{E}[\log N]}{-\mathbb{E}[\log L]} \quad \text{a.s.}$$

其中$N$是分支数，$L$是压缩比。

**证明**：利用大数定律和覆盖论证。对于$\epsilon$-覆盖，期望覆盖数为：
$$\mathbb{E}[N(\epsilon)] \approx \mathbb{E}[N]^n = \epsilon^{-\frac{\mathbb{E}[\log N]}{\mathbb{E}[-\log L]}}$$

其中$n \sim -\log\epsilon / \mathbb{E}[-\log L]$。$\square$

### 2.4 随机测度的遍历性质

**定理 2.4.1（随机测度的遍历定理）** 设$\mu_{\omega}$是遍历RIFS的随机自相似测度，则对于$\mu_{\omega}$-几乎处处的$x$：

$$\lim_{n \to \infty} \frac{1}{n} \sum_{k=0}^{n-1} \phi(T^k x) = \int \phi d\bar{\mu} \quad \text{a.s.}$$

其中$\bar{\mu} = \mathbb{E}[\mu_{\omega}]$是平均测度。

**定理 2.4.2（中心极限定理）** 对于适当的可积函数$\phi$，标准化和收敛于正态分布：

$$\frac{1}{\sqrt{n}} \sum_{k=0}^{n-1} \left(\phi(T^k x) - \int \phi d\bar{\mu}\right) \xrightarrow{d} \mathcal{N}(0, \sigma^2)$$

## 3. 分形上的随机过程

### 3.1 分形布朗运动

**定义 3.1.1（分形布朗运动，FBM）**  Hurst指数为$H \in (0, 1)$的分形布朗运动$B_t^H$是满足以下条件的中心高斯过程：

1. $B_0^H = 0$
2. **自相似性**：$B_{ct}^H \stackrel{d}{=} c^H B_t^H$对所有$c > 0$
3. **协方差**：$\mathbb{E}[B_t^H B_s^H] = \frac{1}{2}(|t|^{2H} + |s|^{2H} - |t-s|^{2H})$

**定理 3.1.2（FBM的分数阶积分表示）** 分形布朗运动有表示：

$$B_t^H = \frac{1}{\Gamma(H + 1/2)} \int_{\mathbb{R}} \left[(t-s)_+^{H-1/2} - (-s)_+^{H-1/2}\right] dW_s$$

其中$W_s$是标准布朗运动。

**定理 3.1.3（FBM的Holder连续性）** FBM几乎必然具有$\alpha$-Holder连续性，对所有$\alpha < H$。

### 3.2 分形扩散过程

**定义 3.2.1（分形扩散）** 分形扩散$X_t$是满足以下随机微分方程的过程：

$$dX_t = \sqrt{2D_{d_s}} dB_t^H$$

其中$D_{d_s}$是谱维$d_s$依赖的扩散系数。

**定理 3.2.2（分形扩散的均方位移）** 分形扩散的均方位移满足：

$$\mathbb{E}[|X_t - X_0|^2] \sim t^{2H} = t^{2/d_s}$$

**证明**：直接计算：
$$\mathbb{E}[|X_t|^2] = 2D_{d_s} \mathbb{E}[|B_t^H|^2] = 2D_{d_s} t^{2H}$$

利用$H = 1/d_s$（对于分形扩散）。$\square$

**定义 3.2.3（反常扩散指数）** 反常扩散指数$\alpha$定义为：

$$\mathbb{E}[|X_t|^2] \sim t^{\alpha}$$

对于正常扩散$\alpha = 1$，对于分形扩散$\alpha = 2/d_s$。

### 3.3 分形上的随机游走

**定义 3.3.1（分形上的简单随机游走）** 设$\mathcal{G}$是分形图，$X_n$是在$\mathcal{G}$上的简单随机游走，转移概率为：

$$P(X_{n+1} = y | X_n = x) = \frac{1}{\deg(x)} \quad \text{如果} \, y \sim x$$

**定理 3.3.2（随机游走的谱维关系）** 分形图上随机游走的返回概率满足：

$$P(X_{2n} = x | X_0 = x) \sim n^{-d_s/2}$$

其中$d_s$是图的谱维。

**定理 3.3.3（随机游走的有效维数）** 随机游走的有效维数等于谱维：

$$d_{\text{eff}} = d_s = \lim_{n \to \infty} \frac{-2\log P(X_{2n} = x | X_0 = x)}{\log n}$$

### 3.4 转移密度的渐近行为

**定理 3.4.1（分形热核估计）** 分形上扩散过程的转移密度（热核）$p(t, x, y)$满足估计：

$$c_1 t^{-d_s/2} \exp\left(-c_2 \left(\frac{|x-y|^{d_w}}{t}\right)^{\frac{1}{d_w-1}}\right) \leq p(t, x, y) \leq c_3 t^{-d_s/2} \exp\left(-c_4 \left(\frac{|x-y|^{d_w}}{t}\right)^{\frac{1}{d_w-1}}\right)$$

其中$d_w$是游走维数（walk dimension）。

**定理 3.4.2（谱维与维数关系）** 对于分形图，维数满足关系：

$$d_s = \frac{2d_f}{d_w}$$

其中$d_f$是Hausdorff维数，$d_w$是游走维数。

## 4. 分形上的随机微分方程

### 4.1 分形随机积分的定义

**定义 4.1.1（分形伊藤积分）** 设$\mathcal{M}$是分形空间，谱维为$d_s$。分形伊藤积分定义为：

$$\int_0^t \sigma(X_s) dB_s^{H} = \lim_{n \to \infty} \sum_{i=0}^{n-1} \sigma(X_{t_i}) (B_{t_{i+1}}^{H} - B_{t_i}^{H})$$

其中极限在$L^2$意义下存在。

**定理 4.1.2（分形伊藤等距）** 对于适应性过程$\sigma(X_t)$：

$$\mathbb{E}\left[\left|\int_0^t \sigma(X_s) dB_s^{H}\right|^2\right] = H(2H-1) \int_0^t \int_0^t |u-v|^{2H-2} \mathbb{E}[\sigma(X_u)\sigma(X_v)] du dv$$

### 4.2 存在唯一性定理

**定理 4.2.1（分形SDE的存在唯一性）** 考虑分形上的随机微分方程：

$$\boxed{dX_t = b(X_t)dt + \sigma(X_t)dB_t^{H}, \quad X_0 = x_0}$$

假设：
1. **Lipschitz条件**：$|b(x) - b(y)| + |\sigma(x) - \sigma(y)| \leq L|x - y|$
2. **线性增长**：$|b(x)| + |\sigma(x)| \leq C(1 + |x|)$

则存在唯一的强解$X_t$。

**证明**：定义算子$\mathcal{T}$：
$$\mathcal{T}(X)_t = x_0 + \int_0^t b(X_s)ds + \int_0^t \sigma(X_s)dB_s^{H}$$

利用分形伊藤等距和Banach不动点定理证明压缩性。$\square$

**定理 4.2.2（解的正则性）** 在额外光滑性假设下，解$X_t$具有$\beta$-Holder连续性，其中$\beta < H$。

### 4.3 分形伊藤公式

**定理 4.3.1（分形伊藤公式）** 设$X_t$满足分形SDE，$f \in C^{2,1}(\mathbb{R} \times [0, T])$，则：

$$\boxed{df(X_t, t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial x}dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}d\langle X \rangle_t}$$

其中二次变差：
$$d\langle X \rangle_t = \sigma(X_t)^2 d\langle B^{H} \rangle_t$$

对于FBM，$\langle B^{H} \rangle_t$具有特殊的分形结构。

**定理 4.3.2（分形Feynman-Kac公式）** 设$u(x, t)$满足：
$$\frac{\partial u}{\partial t} = \mathcal{L}_{d_s}u + V(x)u, \quad u(x, 0) = f(x)$$

其中$\mathcal{L}_{d_s}$是分形生成元，则：
$$u(x, t) = \mathbb{E}\left[f(X_t) \exp\left(\int_0^t V(X_s)ds\right) \bigg| X_0 = x\right]$$

### 4.4 分形Fokker-Planck方程

**定义 4.4.1（分形Fokker-Planck方程）** 概率密度$p(x, t)$满足：

$$\boxed{\frac{\partial p}{\partial t} = -\frac{\partial}{\partial x}(b(x)p) + \frac{1}{2}\frac{\partial^2}{\partial x^2}(\sigma(x)^2 D_{d_s}p)}$$

其中$D_{d_s}$是谱维依赖的分数阶导数算子。

**定理 4.4.2（稳态解的存在性）** 如果漂移项满足细致平衡条件，则稳态解存在：

$$p_{\text{ss}}(x) \propto \exp\left(-\frac{2\Phi(x)}{D_{d_s}}\right)$$

其中$\Phi$是势函数，$b(x) = -\nabla\Phi(x)$。

## 5. 随机谱维理论

### 5.1 谱维的随机涨落

**定义 5.1.1（随机谱维）** 随机谱维$d_s(t, \omega)$是定义在概率空间$(\Omega, \mathcal{F}, \mathbb{P})$上的随机过程，满足：

1. **均值恢复**：$\mathbb{E}[d_s(t)] = \bar{d}_s$
2. **涨落有界**：$d_s(t) \in [d_s^{\min}, d_s^{\max}]$
3. **连续性**：$t \mapsto d_s(t)$几乎必然连续

**定义 5.1.2（谱维的Ornstein-Uhlenbeck模型）** 谱维涨落的OU模型为：

$$dd_s(t) = \theta(\bar{d}_s - d_s(t))dt + \sigma_d dW_t$$

其中$\theta > 0$是回复速率，$\sigma_d > 0$是波动强度。

### 5.2 随机谱维的统计性质

**定理 5.2.1（OU过程的稳态分布）** 随机谱维$d_s(t)$的稳态分布是正态分布：

$$d_s(t) \xrightarrow{d} \mathcal{N}\left(\bar{d}_s, \frac{\sigma_d^2}{2\theta}\right)$$

**证明**：OU过程是高斯马尔可夫过程，其稳态分布由Fokker-Planck方程的稳态解给出。$\square$

**定理 5.2.2（自相关函数）** 随机谱维的自相关函数为：

$$C_{d_s}(\tau) = \text{Cov}(d_s(t), d_s(t+\tau)) = \frac{\sigma_d^2}{2\theta} e^{-\theta|\tau|}$$

### 5.3 随机谱维的遍历定理

**定理 5.3.1（谱维遍历定理）** 对于遍历的随机谱维过程：

$$\lim_{T \to \infty} \frac{1}{T} \int_0^T d_s(t) dt = \bar{d}_s \quad \text{a.s.}$$

**定理 5.3.2（涨落的大偏差原理）** 谱维涨落满足大偏差原理：

$$P\left(\frac{1}{T} \int_0^T d_s(t) dt - \bar{d}_s > a\right) \sim \exp(-T I(a))$$

其中$I(a)$是速率函数。

### 5.4 谱维涨落的关联函数

**定义 5.4.1（多点关联函数）** 谱维涨落的$n$点关联函数定义为：

$$C_n(t_1, \ldots, t_n) = \mathbb{E}\left[\prod_{i=1}^n (d_s(t_i) - \bar{d}_s)\right]$$

**定理 5.4.2（高斯涨落的Wick定理）** 对于高斯型谱维涨落，高阶关联函数分解为两两配对：

$$C_{2n}(t_1, \ldots, t_{2n}) = \sum_{\text{pairings}} \prod_{i<j} C_2(t_i, t_j)$$

## 6. 分形涨落-耗散定理

### 6.1 经典涨落-耗散定理回顾

**定理 6.1.1（经典涨落-耗散定理）** 设$X(t)$是处于热平衡的系统，响应函数$\chi(t)$与关联函数$C(t)$满足：

$$\chi''(\omega) = \frac{\omega}{2k_B T} C(\omega)$$

或用时间域表示：
$$k_B T \frac{d\chi(t)}{dt} = -C(t) \quad \text{对于} \, t > 0$$

### 6.2 分形版本的形式推导

**定义 6.2.1（分形响应函数）** 分形响应函数定义为：

$$\chi_{d_s}(t) = \frac{\delta \langle X(t) \rangle}{\delta h(0)}$$

其中$h$是外场。

**定义 6.2.2（分形关联函数）** 分形关联函数：

$$C_{d_s}(t) = \langle X(t) X(0) \rangle_{d_s} - \langle X \rangle_{d_s}^2$$

其中平均在谱维$d_s$的Gibbs系综中进行。

**定理 6.2.3（分形涨落-耗散定理——形式形式）** 对于分形系统：

$$\chi_{d_s}''(\omega) = \frac{\omega^{d_s/2}}{2k_B T} C_{d_s}(\omega)$$

频率依赖性由谱维修改。

### 6.3 严格数学表述

**定理 6.3.1（分形涨落-耗散定理——严格表述）** 设$\mathcal{H}_{d_s}$是分形系统的Hamilton量，$\rho_{\beta}^{(d_s)} = e^{-\beta \mathcal{H}_{d_s}}/Z$是热态，则对于可观测量$A$和$B$：

$$\boxed{\chi_{AB}^{(d_s)}(\omega) = \frac{1}{2\pi i} \int_{-\infty}^{\infty} \frac{e^{\beta\omega'} - 1}{\omega - \omega' + i\epsilon} C_{AB}^{(d_s)}(\omega') d\omega'}$$

其中关联函数：
$$C_{AB}^{(d_s)}(t) = \text{Tr}_{d_s}\left(\rho_{\beta}^{(d_s)} A(t) B(0)\right)$$

**证明**：利用分形热核和Kubo公式。响应函数可以表示为：
$$\chi_{AB}^{(d_s)}(t) = i\theta(t) \langle [A(t), B(0)] \rangle_{d_s}$$

通过谱表示和分形迹的性质，导出频率域的关系。$\square$

**定理 6.3.2（分形Kubo公式）** 线性响应可以表示为：

$$\langle A(t) \rangle_{d_s} = \langle A \rangle_{d_s}^{(0)} + \int_{-\infty}^t \chi_{AB}^{(d_s)}(t-s) h(s) ds$$

### 6.4 物理应用

**应用 6.4.1（分形介质中的扩散）** 分形涨落-耗散定理预测：

$$D_{d_s}(\omega) \propto \omega^{1-d_s/2}$$

频率依赖的扩散系数。

**应用 6.4.2（分形系统中的热传导）** 热导率$\kappa_{d_s}$与能流关联函数的关系：

$$\kappa_{d_s} = \frac{1}{k_B T^2} \int_0^{\infty} C_J^{(d_s)}(t) dt$$

其中$C_J^{(d_s)}$是热流关联函数。
