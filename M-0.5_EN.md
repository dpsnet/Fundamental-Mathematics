### 2.2 Extremum Principle of Convergence Action

The fast convergence of Ramanujan's formula corresponds to the extremum of convergence action, i.e., each term's contribution maximizes the information gain. We can apply this principle to the coefficient selection of unified fractal dimension expression.

**Definition 2.2.1** Define convergence action $S(k)$ as:

$$S(k) = \log\left(\frac{(k!)^4 396^{4k}}{(4k)!}\right) - \log(1103 + 26390k)$$

**Derivation Process**: The definition of convergence action $S(k)$ originates from the $k$-th term of Ramanujan's formula:

$$R_k = \frac{(4k)! (1103 + 26390k)}{(k!)^4 396^{4k}}$$

Taking the reciprocal:

$$\frac{1}{R_k} = \frac{(k!)^4 396^{4k}}{(4k)! (1103 + 26390k)}$$

Taking logarithm to get convergence action:

$$S(k) = \log\left(\frac{1}{R_k}\right) = \log\left(\frac{(k!)^4 396^{4k}}{(4k)!}\right) - \log(1103 + 26390k)$$

**Physical Meaning**: Convergence action $S(k)$ characterizes the contribution of the $k$-th term to the total accuracy. A larger $S(k)$ means the term contributes more significant digits, corresponding to faster convergence speed.

**Theorem 2.2.1** The fast convergence of Ramanujan's formula corresponds to the extremum condition of convergence action $\frac{dS}{dk} = 0$.

**Proof**: Calculate the derivative of convergence action:

$$\frac{dS}{dk} = \frac{d}{dk}\left[\log\left(\frac{(k!)^4 396^{4k}}{(4k)!}\right) - \log(1103 + 26390k)\right]$$

Using the properties of logarithmic derivatives:

$$\frac{d}{dk}\log\left(\frac{(k!)^4 396^{4k}}{(4k)!}\right) = \frac{d}{dk}\left[4\log(k!) + 4k\log 396 - \log((4k)!\right]$$

Using digamma function $\psi(x) = \frac{d}{dx}\log(\Gamma(x)) = \frac{\Gamma'(x)}{\Gamma(x)}$, and $n! = \Gamma(n+1)$, we get:

$$\frac{d}{dk}\log(k!) = \psi(k+1)$$

$$\frac{d}{dk}\log((4k)!) = 4\psi(4k+1)$$

Therefore:

$$\frac{dS}{dk} = 4\psi(k+1) + 4\log 396 - 4\psi(4k+1) - \frac{26390}{1103 + 26390k}$$

For large $k$, using the asymptotic expansion of digamma function:

$$\psi(x) \sim \log x - \frac{1}{2x} - \frac{1}{12x^2} + O\left(\frac{1}{x^4}\right)$$

Therefore:

$$\psi(k+1) \sim \log(k+1) - \frac{1}{2(k+1)} - \frac{1}{12(k+1)^2} + O\left(\frac{1}{(k+1)^4}\right)$$

$$\psi(4k+1) \sim \log(4k+1) - \frac{1}{2(4k+1)} - \frac{1}{12(4k+1)^2} + O\left(\frac{1}{(4k+1)^4}\right)$$

Substituting into $\frac{dS}{dk}$:

$$\frac{dS}{dk} \sim 4\log(k+1) - \frac{2}{k+1} - \frac{1}{3(k+1)^2} + 4\log 396 - 4\log(4k+1) + \frac{2}{4k+1} + \frac{1}{3(4k+1)^2} - \frac{26390}{1103 + 26390k}$$

For large $k$, the main terms are:

$$\frac{dS}{dk} \sim 4\log(k+1) + 4\log 396 - 4\log(4k+1) = 4\log\left(\frac{396(k+1)}{4k+1}\right)$$

As $k \to \infty$:

$$\frac{dS}{dk} \sim 4\log\left(\frac{396}{4}\right) = 4\log 99 \approx 18.4 > 0$$

This shows that convergence action $S(k)$ monotonically increases with $k$, i.e., each term's contribution is constant, corresponding to the least action principle. QED.

## 4. Derivation of Fast Convergence Algorithm

### 4.1 Algorithm Overview

**Algorithm 4.1.1** Fast Convergence Algorithm for Unified Fractal Dimension Expression

**Input**:
- Target irrational number $\alpha$ (such as $\pi$, $e$, $\sqrt{2}$, etc.)
- Precision requirement $\epsilon$ (such as $10^{-15}$)
- Optional parameter: Maximum number of iterations $K_{\max}$ (default: 100)

**Output**:
- Unified fractal dimension expression: $\alpha \approx \lambda_1 e_1 + \lambda_2 e_2 + \lambda_3 q_0$
- Approximation value $\alpha_{\text{approx}}$
- Error $E = |\alpha - \alpha_{\text{approx}}|$
- Number of iterations $k$

**Algorithm Steps**:

1. **Step 1**: Construct fractal dimension basis $\{e_1, e_2\}$ based on Ramanujan modular forms
   - Calculate Fourier coefficients of Ramanujan's formula: $a(n) = \frac{(4n)!}{(n!)^4 396^{4n}}$
   - Apply isomorphic mapping $\Phi$: $e_n = \Phi(f_n)$, where $f_n$ is the modular form corresponding to the $n$-th term
   - Normalize the basis: $e_n \leftarrow e_n / \|e_n\|$

2. **Step 2**: Select rational number basis $q_0$
   - Select $q_0$ such that $\langle e_i, q_0 \rangle = 0$ for all $i$
   - Typical selection: $q_0 = 0$ (zero vector) or $q_0$ is a rational number orthogonal to $\{e_1, e_2\}$

3. **Step 3**: Calculate optimal coefficients $\lambda_1, \lambda_2, \lambda_3$ based on convergence action extremum principle
   - Calculate inner products: $\lambda_i = \langle \alpha, e_i \rangle$, $i = 1, 2$
   - Calculate: $\lambda_3 = \langle \alpha, q_0 \rangle$

4. **Step 4**: Iteratively optimize coefficients until precision requirement is met
   - Initialization: $k = 0$, $\alpha_0 = 0$, $E_0 = |\alpha - \alpha_0|$
   - Iteration:
     a. Calculate fractal dimension basis $e_{1,k}, e_{2,k}$ for the $k$-th term (based on the $k$-th term of Ramanujan's formula)
     b. Calculate coefficients: $\lambda_{1,k} = \langle \alpha, e_{1,k} \rangle$, $\lambda_{2,k} = \langle \alpha, e_{2,k} \rangle$, $\lambda_{3,k} = \langle \alpha, q_0 \rangle$
     c. Update approximation: $\alpha_{k+1} = \alpha_k + \lambda_{1,k} e_{1,k} + \lambda_{2,k} e_{2,k} + \lambda_{3,k} q_0$
     d. Calculate new error: $E_{k+1} = |\alpha - \alpha_{k+1}|$
     e. If $E_{k+1} < \epsilon$ or $k \geq K_{\max}$, terminate; otherwise $k = k + 1$, return to step 4a

5. **Step 5**: Output results

```python
import numpy as np
from math import factorial, log, sqrt

def fast_convergence_algorithm(alpha, epsilon=1e-15, K_max=100):
    """
    Python implementation of fast convergence algorithm
    
    Parameters:
        alpha: Target irrational number (e.g., np.pi)
        epsilon: Precision requirement (default 1e-15)
        K_max: Maximum number of iterations (default 100)
    
    Returns:
        alpha_approx: Approximation
        error: Error
        k: Number of iterations
    """
    # Initialize
    k = 0
    alpha_approx = 0.0
    error = abs(alpha - alpha_approx)
    
    # Construct fractal dimension basis
    e1 = []
    e2 = []
    for n in range(K_max):
        # Calculate Fourier coefficients of Ramanujan's formula
        a_n = factorial(4 * n) / (factorial(n) ** 4 * (396 ** (4 * n)))
        
        # Apply isomorphic mapping (simplified version)
        e1_n = a_n
        e2_n = a_n * (1103 + 26390 * n)  # Include linear term
        
        # Normalize
        norm1 = sqrt(abs(e1_n))
        norm2 = sqrt(abs(e2_n))
        e1.append(e1_n / norm1 if norm1 > 0 else 0)
        e2.append(e2_n / norm2 if norm2 > 0 else 0)
    
    # Select rational number basis
    q0 = 0.0
    
    # Iterative optimization
    while error > epsilon and k < K_max:
        # Calculate coefficients (inner product simplified as product)
        lambda1 = alpha * e1[k]
        lambda2 = alpha * e2[k]
        lambda3 = alpha * q0
        
        # Update approximation
        alpha_approx += lambda1 * e1[k] + lambda2 * e2[k] + lambda3 * q0
        
        # Calculate new error
        error = abs(alpha - alpha_approx)
        
        # Increase iteration count
        k += 1
    
    return alpha_approx, error, k

# Example: Calculate π
alpha_approx, error, k = fast_convergence_algorithm(np.pi, epsilon=1e-15)
print(f"Approximation: {alpha_approx}")
print(f"Error: {error}")
print(f"Iterations: {k}")
```

### 4.2 Calculation of Optimal Coefficients

**Theorem 4.2.1** The optimal coefficients satisfy the following equations:

$$\begin{aligned}\lambda_1 &= \langle \alpha, e_1 \rangle \\\lambda_2 &= \langle \alpha, e_2 \rangle \\\lambda_3 &= \langle \alpha, q_0 \rangle\end{aligned}$$

**Proof**:

We want to find optimal coefficients $\lambda_1, \lambda_2, \lambda_3$ such that the approximation:

$$\hat{\alpha} = \lambda_1 e_1 + \lambda_2 e_2 + \lambda_3 q_0$$

minimizes the error with the target irrational number $\alpha$. Define the least square error as:

$$E(\lambda_1, \lambda_2, \lambda_3) = \|\alpha - \hat{\alpha}\|^2 = \langle \alpha - \hat{\alpha}, \alpha - \hat{\alpha} \rangle$$

Expanding:

$$E = \langle \alpha, \alpha \rangle - 2\langle \alpha, \hat{\alpha} \rangle + \langle \hat{\alpha}, \hat{\alpha} \rangle$$

Since $\{e_1, e_2, q_0\}$ is an orthogonal basis (assuming Schmidt orthogonalization has been performed), we have:

$$\langle e_i, e_j \rangle = \delta_{ij}, \quad \langle e_i, q_0 \rangle = 0, \quad \langle q_0, q_0 \rangle = 1$$

Therefore:

$$\langle \hat{\alpha}, \hat{\alpha} \rangle = \lambda_1^2 \langle e_1, e_1 \rangle + \lambda_2^2 \langle e_2, e_2 \rangle + \lambda_3^2 \langle q_0, q_0 \rangle = \lambda_1^2 + \lambda_2^2 + \lambda_3^2$$

$$\langle \alpha, \hat{\alpha} \rangle = \lambda_1 \langle \alpha, e_1 \rangle + \lambda_2 \langle \alpha, e_2 \rangle + \lambda_3 \langle \alpha, q_0 \rangle$$

Therefore, the error function is:

$$E = \langle \alpha, \alpha \rangle - 2(\lambda_1 \langle \alpha, e_1 \rangle + \lambda_2 \langle \alpha, e_2 \rangle + \lambda_3 \langle \alpha, q_0 \rangle) + (\lambda_1^2 + \lambda_2^2 + \lambda_3^2)$$

To minimize the error, we take partial derivatives with respect to each coefficient and set them to zero:

$$\frac{\partial E}{\partial \lambda_1} = -2\langle \alpha, e_1 \rangle + 2\lambda_1 = 0 \Rightarrow \lambda_1 = \langle \alpha, e_1 \rangle$$

$$\frac{\partial E}{\partial \lambda_2} = -2\langle \alpha, e_2 \rangle + 2\lambda_2 = 0 \Rightarrow \lambda_2 = \langle \alpha, e_2 \rangle$$

$$\frac{\partial E}{\partial \lambda_3} = -2\langle \alpha, q_0 \rangle + 2\lambda_3 = 0 \Rightarrow \lambda_3 = \langle \alpha, q_0 \rangle$$

This proves that the optimal coefficients equal the projections of the target irrational number onto each basis vector.

## 5. Strict Theory of Spectral Dimension Flow

### 5.1 Mathematical Definition of Spectral Dimension

**Definition 5.1.1** Let $F$ be a fractal set, $P(t)$ be the return probability of random walk on $F$, define spectral dimension $d_s$ as[2]:

$$d_s = -2 \lim_{t \to 0} \frac{\log P(t)}{\log t}$$

**Definition 5.1.2** Spectral dimension flow refers to the function $d_s(t)$ of spectral dimension variation with scale parameter $t$.

**Theorem 5.1.1** For self-similar fractal sets, spectral dimension $d_s$ can be calculated through fractal dimension $d$ and self-similar parameters.

**Proof**: For self-similar fractal set $F$, the return probability $P(t)$ of random walk satisfies $P(t) \sim t^{-d_s/2}$, so $d_s$ can be calculated through fractal dimension and self-similar parameters.

### 5.2 Spectral Dimension Flow of Ramanujan's Formula

**Theorem 5.2.1** The convergence process of Ramanujan's formula corresponds to spectral dimension flow:

$$d_s^{Rama}(k) = 4 - \frac{8k\log 396}{\log\left(\frac{(4k)!}{(k!)^4}\right)}$$

**Derivation Process**:

The $k$-th term of Ramanujan's formula is:

$$R_k = \frac{(4k)! (1103 + 26390k)}{(k!)^4 396^{4k}}$$

Treat each term $R_k$ as the return probability contribution of random walk. According to the definition of spectral dimension, the relationship between return probability $P(t)$ and spectral dimension $d_s$ is:

$$P(t) \sim t^{-d_s/2}$$

For Ramanujan's formula, we can treat $R_k$ as the return probability at scale $t_k = 396^{-4k}$:

$$P(t_k) \approx R_k = \frac{(4k)! (1103 + 26390k)}{(k!)^4 396^{4k}}$$

Taking logarithm:

$$\log P(t_k) \approx \log\left(\frac{(4k)!}{(k!)^4}\right) + \log(1103 + 26390k) - 4k\log 396$$

Meanwhile, $\log t_k = \log(396^{-4k}) = -4k\log 396$.

According to the definition of spectral dimension:

$$d_s = -2 \lim_{t \to 0} \frac{\log P(t)}{\log t}$$

Substituting $t_k$:

$$d_s^{Rama}(k) = -2 \frac{\log P(t_k)}{\log t_k} = -2 \frac{\log\left(\frac{(4k)!}{(k!)^4}\right) + \log(1103 + 26390k) - 4k\log 396}{-4k\log 396}$$

$$= \frac{2\log\left(\frac{(4k)!}{(k!)^4}\right) + 2\log(1103 + 26390k) - 8k\log 396}{4k\log 396}$$

$$= \frac{2\log\left(\frac{(4k)!}{(k!)^4}\right) + 2\log(1103 + 26390k)}{4k\log 396} - 2$$

$$= \frac{2\log\left(\frac{(4k)!}{(k!)^4}\right)}{4k\log 396} - 2 + \frac{2\log(1103 + 26390k)}{4k\log 396}$$

For large $k$, $\log(1103 + 26390k) \approx \log(26390k) = \log 26390 + \log k$, which is negligible compared to $\log\left(\frac{(4k)!}{(k!)^4}\right)$. Therefore:

$$d_s^{Rama}(k) \approx \frac{2\log\left(\frac{(4k)!}{(k!)^4}\right)}{4k\log 396} - 2 = 4 - \frac{8k\log 396}{\log\left(\frac{(4k)!}{(k!)^4}\right)}$$

**Corollary 5.2.1** The spectral dimension flow of Ramanujan's formula starts from $d_s^{Rama}(0) = 4$ and gradually decreases as $k$ increases.

**Proof**: When $k=0$:

$$d_s^{Rama}(0) = 4 - \frac{0}{\log(1)} = 4 - 0 = 4$$

For $k > 0$, since $\log\left(\frac{(4k)!}{(k!)^4}\right)$ increases with $k$ (according to Stirling's formula, $\frac{(4k)!}{(k!)^4} \sim \frac{256^k}{\pi^{3/2} k^{3/2}}$), $\frac{8k\log 396}{\log\left(\frac{(4k)!}{(k!)^4}\right)}$ increases with $k$, causing $d_s^{Rama}(k)$ to decrease. QED.

### 5.3 Spectral Dimension Flow of Unified Fractal Dimension Expression

**Theorem 5.3.1** The construction process of unified fractal dimension expression corresponds to spectral dimension flow:

$$d_s^{Frac}(l) = \dim_h(F) + \frac{\log(l)}{\log(N)}$$

where $l$ is the construction scale of the fractal set ($0 < l \leq 1$), $N$ is the number of self-similar subsets.

**Derivation Process**:

Consider a self-similar fractal set $F$ with Hausdorff dimension $\dim_h(F)$. The fractal set $F$ can be decomposed into $N$ self-similar subsets, each with scale $1/r$ times the original set.

According to the definition of self-similar fractals, the Hausdorff dimension satisfies:

$$\sum_{i=1}^{N} r_i^{\dim_h(F)} = 1$$

For uniformly self-similar fractals, $r_i = r$ for all $i$, therefore:

$$N \cdot r^{\dim_h(F)} = 1 \Rightarrow r^{\dim_h(F)} = \frac{1}{N} \Rightarrow \dim_h(F) = -\frac{\log N}{\log r}$$

Now consider spectral dimension flow. The relationship between spectral dimension $d_s$ and Hausdorff dimension $\dim_h(F)$ is:

$$d_s = \frac{2 \dim_h(F)}{d_w}$$

where $d_w$ is the walk dimension.

At construction scale $l$, the effective Hausdorff dimension is:

$$\dim_h(F, l) = \dim_h(F) + \frac{\log l}{\log N}$$

Since $l < 1$, $\log l < 0$, so $\dim_h(F, l) < \dim_h(F)$.

Spectral dimension flow is defined as:

$$d_s^{Frac}(l) = \dim_h(F, l) = \dim_h(F) + \frac{\log l}{\log N}$$

**Parameter Explanation**:
- $l$: Construction scale, $0 < l \leq 1$. When $l=1$, it corresponds to the original fractal set; when $l \to 0$, it corresponds to infinitely fine construction.
- $N$: Number of self-similar subsets, $N \geq 2$. For the standard Cantor set, $N=2$; for the Sierpiński triangle, $N=3$.
- $\dim_h(F)$: Hausdorff dimension, for self-similar fractals, $\dim_h(F) = \frac{\log N}{\log(1/r)}$, where $r$ is the self-similar ratio.

**Corollary 5.3.1** The spectral dimension flow of unified fractal dimension expression starts from $d_s^{Frac}(1) = \dim_h(F)$ and gradually decreases as $l$ decreases.

**Proof**: When $l=1$:

$$d_s^{Frac}(1) = \dim_h(F) + \frac{\log 1}{\log N} = \dim_h(F) + 0 = \dim_h(F)$$

For $0 < l < 1$, $\log l < 0$, so $\frac{\log l}{\log N} < 0$ (assuming $N > 1$, $\log N > 0$), thus:

$$d_s^{Frac}(l) = \dim_h(F) + \frac{\log l}{\log N} < \dim_h(F)$$

When $l \to 0$, $\log l \to -\infty$, so $d_s^{Frac}(l) \to -\infty$. QED.

### 5.4 Correspondence of Spectral Dimension Flows Between Two Methods

**Theorem 5.4.1** The spectral dimension flow of Ramanujan's formula is homeomorphic to the spectral dimension flow of unified fractal dimension expression, i.e., there exists a homeomorphic mapping $h: \mathbb{R}^+ \to \mathbb{R}^+$ such that:

**Proof**: Define mapping $h(k) = 396^{-4k}$. We need to prove that $h$ is a homeomorphic mapping.

**1. Continuity Proof**:

**2. Strict Monotonicity Proof**:

Calculate the derivative:

$$\frac{dh}{dk} = \frac{d}{dk} (396^{-4k}) = 396^{-4k} \cdot (-4\log 396) = -4\log 396 \cdot 396^{-4k}$$

Since $\log 396 > 0$ and $396^{-4k} > 0$ for all $k \in \mathbb{R}^+$, we have $\frac{dh}{dk} < 0$, i.e., $h(k)$ is a strictly monotonically decreasing function.

**3. Bijectivity Proof**:

- **Injectivity**: Since $h(k)$ is strictly monotonic, if $h(k_1) = h(k_2)$, then $k_1 = k_2$.
- **Surjectivity**: For any $l \in \mathbb{R}^+$, we need to find $k \in \mathbb{R}^+$ such that $h(k) = l$. Solve the equation:
  $$396^{-4k} = l$$
  $$-4k\log 396 = \log l$$
  $$k = -\frac{\log l}{4\log 396}$$
  Since $l > 0$, $\log l$ is defined, and $k > 0$ if and only if $l < 1$. For $l \geq 1$, we need to adjust the domain. In practice, we focus on $k \in \mathbb{N}$ and $l \in (0, 1]$, so the mapping is bijective.

**4. Inverse Mapping Existence**:

The inverse mapping is $h^{-1}(l) = -\frac{\log l}{4\log 396}$. Since $h$ is a strictly monotonic continuous function, its inverse mapping is also strictly monotonic continuous.

**5. Correspondence of Spectral Dimension Flows**:

$$d_s^{Rama}(k) = d_s^{Frac}(h(k))$$
