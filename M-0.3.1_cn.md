$$e = \lambda_{e1} e_1 + \lambda_{e2} e_2 + \lambda_{e3} q_0$$

$$\lambda_{e1} e_1 + \lambda_{e2} e_2 + \lambda_{e3} q_0 \approx \lambda_{\pi1} e_1 + \lambda_{\pi2} e_2 + \lambda_{\pi3} q_0 - \delta$$

$$(\lambda_{e1} - \lambda_{\pi1}) e_1 + (\lambda_{e2} - \lambda_{\pi2}) e_2 + (\lambda_{e3} - \lambda_{\pi3}) q_0 + \delta \approx 0$$

由于 $e_1, e_2, q_0$ 是正交基，上式成立当且仅当各系数近似为零：

$$\lambda_{e1} \approx \lambda_{\pi1}$$
$$\lambda_{e2} \approx \lambda_{\pi2}$$

import mpmath

# 设置高精度（100位小数）
mpmath.mp.dps = 100

class UnifiedFractalDimensionExpression:
    """统一分形维数表达式类"""
    
    def __init__(self, e1, e2, q0):
        """
        初始化统一分形维数表达式
        
        参数:
        e1, e2: 正交分形维数基
        q0: 有理数基
        """
        self.e1 = e1
        self.e2 = e2
        self.q0 = q0
    
    def express(self, alpha, max_iter=100, tol=1e-50):
        """
        将实数alpha表示为统一分形维数表达式：alpha = λ1*e1 + λ2*e2 + λ3*q0
        使用正交投影法求解有理数系数
        
        参数:
        alpha: 目标实数
        max_iter: 最大迭代次数
        tol: 收敛容差
        
        返回:
        λ1, λ2, λ3: 高精度浮点系数（实际应用中需要约束为有理数）
        """
        # 初始化系数
        lambda1 = mpmath.mpf(0)
        lambda2 = mpmath.mpf(0)
        lambda3 = mpmath.mpf(0)
        
        # 学习率
        eta = mpmath.mpf(0.1)
        
        for i in range(max_iter):
            # 计算当前近似值
            approx = lambda1 * self.e1 + lambda2 * self.e2 + lambda3 * self.q0
            # 计算误差
            error = alpha - approx
            
            # 检查收敛条件
            if abs(error) < tol:
                break
            
            # 正交投影更新系数
            lambda1 += eta * error * self.e1
            lambda2 += eta * error * self.e2
            lambda3 += eta * error * self.q0
        
        # 直接返回高精度浮点系数
        return lambda1, lambda2, lambda3
    
    def compute(self, lambda1, lambda2, lambda3):
        """使用统一分形维数表达式计算值"""
        return lambda1 * self.e1 + lambda2 * self.e2 + lambda3 * self.q0
