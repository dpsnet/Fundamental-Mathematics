# 统一场理论基础数学

本仓库包含统一场理论的基础数学理论和实现，专注于M-0系列的理论模块。

## 仓库结构

### 核心模块
- **M-0.1**：分形维数的基本理论与无理数表示
- **M-0.2**：内积空间与正交组合表示理论
- **M-0.3**：统一分形维数表达式与拉马努金公式映射
  - M-0.3.1：基于统一分形维数表达式的e与π-δ接近关系理论分析
  - M-0.3.2：拉马努金公式与e和π-δ接近关系的深入研究
  - M-0.3.3：基于内积空间正交组合表示的实验可行性分析
  - M-0.3.4：有理系数与项的量化理论深入研究
  - M-0.3.5：谱维流动与高维时空的四维实现
- **M-0.4**：素数生成函数与模形式
- **M-0.5**：快速收敛算法与谱维流动
- **M-0.6**：哥德巴赫猜想的分形几何解释
- **M-0.7**：谱维流动理论的深入研究
- **M-0.8**：4维闵可夫斯基时空的基础推导
- **M-0.9**：PTE问题与分形几何联系
  - M-0.9.2：模形式理论与PTE解生成函数研究
  - M-0.9.3：谱维流动与PTE解尺度变化模型研究
  - M-0.9.4：基于L-BFGS-B的高效PTE解搜索算法设计
  - M-0.9.5：模形式生成的PTE解系统构建研究
  - M-0.9.6：素数PTE解的分布规律分析
  - M-0.9.7：PTE解分布规律研究框架总结
- **M-0.10~M-0.13**：扩展理论模块
- **M-0.14**：时空维度的起源 - 4维的数学必然性
  - M-0.14.1：高能极限下谱维扩展的数学理论
  - M-0.14.2：时间量化的数学理论
  - M-0.14.3：多时间不稳定性的数学理论
  - M-0.14.4：替代4维时空形式的数学分析

### 实现文件
- **源代码**：用于数学验证的Python实现
- **数据**：高精度数值数据与验证结果
- **图表**：分形维数与数学概念的可视化
